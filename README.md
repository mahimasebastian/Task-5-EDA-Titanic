The dataset contains passenger information such as:

- Passenger Class
- Age
- Gender
- Fare

- Number of Parents/Children
- Embarked Port

---

## 📈 Key Findings

- Female passengers had a significantly higher survival rate than males.
- First-class passengers were more likely to survive.
- Higher ticket fares showed a positive relationship with survival.
- Most passengers belonged to the third passenger class.
- The **Fare** column contained several outliers.
- Missing values were mainly found in the **Age** column and were handled using the median.
